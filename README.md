This is a SQL Database Project for Car Dealership. 
It implements:

1. Database Design:
2. Primary Key, Foreign Key, Many-to-Many Relationships, Indexes.

3. Data Manipulation:
4.  Insert Operations, Update Operations, Delete Operations.

5. Data Retrieval and Queries:
6. Aggregate functions, such as COUNT, SUM, AVG, etc. Pagination, using clauses like OFFSET, FETCH NEXT, TOP, etc. GROUP BY, ORDER BY clauses.

7. Table Joins:
8. 2 Table Join, 3 Table Join, Create View.

9. Procedures/Functions/Triggers:
10. SQL Stored procedure
11. SQL trigger
