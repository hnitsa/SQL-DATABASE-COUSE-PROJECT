This is a SQL Database Project for Car Dealership. 
It implements:

1. Database Design
Create a Database: Primary Key, Foreign Key, Many-to-Many Relationships, Indexes.

2. Data Manipulation 
Insert Operations, Update Operations, Delete Operations.

3. Data Retrieval and Queries
Aggregate functions, such as COUNT, SUM, AVG, etc. Pagination, using clauses like OFFSET, FETCH NEXT, TOP, etc. GROUP BY, ORDER BY clauses.

4. Table Joins
2 Table Join, 3 Table Join, Create View.

6. Procedures/Functions/Triggers
SQL Stored procedure 
SQL trigger
